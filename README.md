# Powerful Calculator App

A feature-rich calculator app built with React Native and Expo SDK 48.

## Features

### Calculator Modes
- **Standard**: Basic arithmetic operations
- **Scientific**: Trigonometric functions, logarithms, powers, roots
- **Programmer**: Binary, octal, decimal, hexadecimal with bitwise operations
- **Date Calculator**: Calculate differences and add/subtract days

### Unit Converter (13 Categories)
- Currency, Volume, Length, Weight & Mass, Temperature, Energy, Area, Speed, Time, Power, Data, Pressure, Angle

### UI Features
- Dark/light theme support
- Smooth Reanimated 2 animations
- Haptic feedback
- Calculation history
- Responsive design

## Setup

```bash
npm install
npx expo start
```
